FST UMS – Staff Busyness Index Calculator

Single-file tool (GitHub Pages). Admin can lock weights and scoring scheme; staff only enter activities.
Mode: Staff Settings locked

Staff Profile

All fields are required. Staff ID example: 250505-02025.
Selecting category will auto-apply ON/OFF sections and default weights.

Section Applicability (ON/OFF)

Turn ON only relevant sections. The calculator will automatically re-normalize weights using ON sections only. (This supports Academic Staff, Administrative Staff, and Laboratory Staff fairly.)
Auto from Category

Settings (Weights & Scoring Scheme) – Admin Only

Locked by default. Admin must unlock using password to edit. Baseline weights must sum to 1.00.
Important: During calculation, the system uses only ON sections and re-normalizes weights automatically. Baseline weights (admin) must still sum to 1.00.
Admin password location: At the very top of this index.html, search for ADMIN_PASSWORD.

Teaching

Credit is auto-extracted from the last digit of course code (e.g., SJ24102 → credit=2). Contact hours are auto-computed.
Course Code * Course Name * Credit Students * Lecture * Tutorial * Laboratory * Field * Contact hours Delete

Supervision

Include active supervision load for the reporting period (Undergraduate, Master, PhD).
Level * Student / Project * Role * Status * Start * End * Delete

Research (Grants / Projects)

Specify grant level, status, role, amount and duration.
Grant Code * Title * Level * Status * Role * Amount (RM) * Start * End * Delete

Publication

If ON, record publication output for the reporting period.
Year * Title * Type * Indexed? * Role * Start * End * Delete

Administration

Include appointments (official/ad-hoc), level, duration (months) and meeting workload.
Position * Type * Level * Months * Meetings * Notes * Start * End * Delete
Tip: For conference committee work, use Position = “Conference Committee” and describe preparation tasks in Notes.

Services

Service to university / industry / community. Include start and end dates.
Service Type * Activity * Role * Hours * Participants * Start * End * Delete

Laboratory Operations

For laboratory staff or lab-support work. Scored using (volume + hours) × complexity × urgency.
Type * Description * Volume * Complexity * Urgency * Hours * Start * End * Delete
